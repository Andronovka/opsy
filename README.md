# Операционные системы
## Задание 1. Простая криптографическая библиотека

Создана простая криптографическая библиотека с шифрованием через XOR.

Для запуска указать путь библиотеки.

## Задание 2. Простой файловый копировщик с шифрованием

Создано консольное приложение secure_copy для копирования файлов с одновременным шифрованием содержимого.

Для запуска указать путь библиотеки.

## Задание 3. Защита от конфликтов при одновременном копировании

Расширен функциональность программы secure_copy для одновременной обработки нескольких файлов с использованием механизмов синхронизации.

## Задание 4. Оптимизация копирования
Реализовано два режима работы программы — последовательный и параллельный — с автоматическим выбором наиболее эффективного режима в зависимости от количества обрабатываемых файлов
